# Kanban App

A feature-first React kanban board built with Vite, Tailwind CSS, and dnd-kit.

## Stack

- React 19 + Vite 8
- Tailwind CSS (token-driven theme layer)
- dnd-kit for drag and drop
- react-day-picker for themed due-date picking

## Scripts

- `npm run dev` - start local development server
- `npm run build` - create production build
- `npm run preview` - preview production build
- `npm run lint` - run ESLint

## Project Structure

- `src/features` - board, column, and task feature modules
- `src/components/ui` - reusable UI primitives
- `src/styles/tokens.css` - semantic theme tokens and shared component classes
- `src/lib` - utility functions
- `src/hooks` - reusable hooks
